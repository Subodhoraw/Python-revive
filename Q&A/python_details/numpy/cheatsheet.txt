# ðŸ“Œ SECTION 3 â€” **Meshgrid (for plotting & coordinate grids)**

```python
x = np.array([1, 2, 3])
y = np.array([4, 5])
X, Y = np.meshgrid(x, y)
```

### Output:

**X:** repeats x across rows

```
[[1 2 3]
 [1 2 3]]
```

**Y:** repeats y across columns

```
[[4 4 4]
 [5 5 5]]
```

Used for:

* heatmaps
* 3D surfaces
* contour plots
* evaluating f(x,y)

* used for ML weight initialization

# ðŸ“Œ SECTION 5 â€” **Vectorization (Performance Optimization)**

Vectorization = replace Python loops with fast NumPy operations.

## âœ” Slow loop

```python
for i in range(len(arr)):
    arr[i] = arr[i] * 2
```

## âœ” Vectorized

```python
arr = arr * 2
```

Much faster due to C-level operations.

## âœ” Vectorized math

```python
arr = np.log(arr)
res = arr1 + arr2
res = arr * arr
```

## âœ” Boolean vectorization

```python

```

## âœ” np.where (vectorized if-else)

```python

```

## âœ” Broadcasting

```python

A + np.array([1,2,3])          # row-wise
A + np.array([[1],[2],[3]])    # column-wise
```
